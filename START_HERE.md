# 🎯 从这里开始

## 🚀 一键启动命令

### macOS / Linux
```bash
./start.sh
```

### Windows
```bash
start.bat
```

### 或使用 npm
```bash
npm start
```

---

## 📍 启动后访问

- **官网**: http://localhost:3001
- **管理后台**: http://localhost:3001/admin
