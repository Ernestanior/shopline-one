# 快速测试指南

## 🚀 5分钟快速测试

### 步骤1：检查服务器（30秒）

```bash
node check-servers.js
```

如果服务器未运行，启动它们：

**终端1 - 后端：**
```bash
node server/index.js
```

**终端2 - 前端：**
```bash
cd client && npm start
```

### 步骤3：浏览器测试（3分钟）

1. **打开浏览器** → http://localhost:3001

2. **注册账户**
   - 点击右上角"登录"
   - 切换到"注册"
   - 输入邮箱和密码
   - 点击"注册"

3. **添加地址**
   - 点击"我的账户"
   - 点击"收货地址"标签
   - 点击"+ 添加新地址"
   - 填写表单
   - 勾选"设为默认地址"
   - 点击"保存"
   - ✅ 验证：地址显示在列表中

4. **添加支付方式**
   - 点击"支付方式"标签
   - 点击"+ 添加支付方式"
   - 填写：
     - 卡号：4242 4242 4242 4242
     - 持卡人：测试用户
     - 有效期：12/2028
     - CVV：123
   - 勾选"设为默认支付方式"
   - 点击"保存"
   - ✅ 验证：支付方式显示，只显示后4位

5. **测试自动填充**
   - 访问商品页面
   - 添加商品到购物车
   - 进入结账
   - ✅ 验证：邮箱已填充，显示"已登录"提示
   - 点击"Continue"
   - ✅ 验证：地址已填充，显示"使用默认地址"提示
   - 点击"Continue"
   - ✅ 验证：支付方式已填充

6. **完成订单**
   - 完成结账流程
   - 返回"我的账户" → "订单历史"
   - ✅ 验证：订单显示在列表中

## 🐛 常见问题

### 问题1：服务器未运行
**解决：** 运行 `node check-servers.js` 检查，然后启动服务器

### 问题2：自动化测试失败
**解决：** 确保服务器运行，等待几秒后重试

### 问题3：无法添加地址
**解决：** 检查是否已登录，打开浏览器控制台查看错误

### 问题4：结账没有自动填充
**解决：** 确保已添加默认地址和支付方式
